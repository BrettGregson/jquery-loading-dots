# jQuery Loading Dots


![Alt text](http://i.imgur.com/CXv1R2y.gif "Loading")


"Animated" jQuery loading dots. Works with centered text without changing the centered texts offset as the dots appear.

Useage:

<pre>
$("#loading").dots({
	"speed"			: 50,
	"numberOfDots"	: 10,
	"loadingWord"	: "Loading",
	"dotCharacter"	: ".",
});
</pre>
